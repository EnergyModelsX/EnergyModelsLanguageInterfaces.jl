This package is an extension to the [`EnergyModelsX`](https://github.com/EnergyModelsX) (EMX) framework.
The aim of this package is to provide utilities for EMX.

In addition, some of the features may be incorporated into `EnergyModelsBase` eventually.

> **Note**
> The package is currently under development. Expect braking changes.
